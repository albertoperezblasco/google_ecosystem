account_tree Agent Development Kit Workflow Code Agent Development Kit Open, modular framework for agent orchestration

Agent Development Kit is Google's code-first framework for developing and orchestrating agentic architectures. It is optimized for Gemini and Google Cloud, but documented as model-agnostic, deployment-agnostic, and compatible with other frameworks. It is the main build surface before deployment to Vertex AI Agent Engine.

Agent Designer is a low-code visual designer in the Google Cloud console. It lets you model main agents and subagents, configure tools, preview behavior, and then hand off generated code to Agent Development Kit for deeper development and deployment.

deployed_code Vertex AI Agent Engine Managed Runtime Vertex AI Agent Engine Production runtime for deploying, managing, and scaling agents

Agent Engine is the managed runtime in the suite. It includes runtime services plus sessions, Memory Bank, Code Execution, Example Store, evaluation, observability, and enterprise governance features. Google also positions Agent Starter Pack as the streamlined template path for running production agent workflows on this runtime.

Agent Garden is the discovery layer inside Vertex AI Agent Builder. Google documents it as a preview library where you can browse Google-published end-to-end agents and individual tools, then deploy agents or reuse the tools inside Agent Development Kit-based builds.
