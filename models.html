Models Atlas

Grouped by the work they are built for.

This page turns the merged model inventory into a navigable map. Every task group lives as its own constellation in the shared universe, and every model keeps its status, family, official description and source URLs visible in hover cards and detailed cards below.

How to use it

Hover a model bubble to read a description sourced from the linked official URL whenever the documentation exposes one clearly. Scroll or drag across the universe, use Ctrl/Cmd plus wheel to zoom, and jump between clusters with the quick navigation chips.

Quick Navigation

Jump straight into a model family.

The map uses the same task groups defined in extracted_data.md. Each chip recenters the universe on that cluster and keeps the detailed sections below aligned with the same grouping.

Navigable Universe

One shared map, many grouped constellations.

Every task group is rendered as its own cluster inside a single scrollable universe. Click a model to jump to its detailed card, or hover to open the descriptive spec panel with URLs.
