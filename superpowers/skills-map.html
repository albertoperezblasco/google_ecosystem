Lectura rapida

  • using-google-genai-refresh no edita contenido: fuerza a empezar por el proceso correcto.
  • La fuente local obligatoria es sourceEcosystemGoogle/google-genai-refresh-source.md.
  • Discovery va primero y ahora corre via subagente: antes de editar hay que confirmar si la surface existe y cual es su URL oficial.
  • Si el usuario pide ejecutar todo el flujo o rehacer el refresh completo, eso implica siempre lanzar el Topic Gap Discovery subagent.
  • Lifecycle va despues: primero se decide si algo esta activo, en preview o deprecado; luego se reescribe.
  • La actualizacion toca cuatro destinos: cards/, google-services-by-topic.md, google_ecosystem.md y este HTML.
  • La verificacion final no es opcional: evita cards huerfanas, listas activas con deprecados y nombres incoherentes.

Regla base

Este flujo no persigue "novedad por novedad". Persigue catalogo util, verificable y mantenible.

No se edita desde memoria.
Primero fuente oficial, despues lifecycle, despues propagacion al repo.

Atajos por escenario

Secuencias cortas para los escenarios mas habituales del repo.

Nueva surface oficial

Cuando Google documenta un producto, framework o tool nuevo y hay que incorporarlo al catalogo.

using-google-genai-refresh -> Topic Gap Discovery subagent -> tracking-google-genai-lifecycle -> updating-google-genai-catalog -> verifying-google-genai-refresh

Deprecation o shutdown

Cuando algo que ya estaba en el repo deja de ser ruta recomendada o tiene fecha de retirada.

using-google-genai-refresh -> tracking-google-genai-lifecycle -> updating-google-genai-catalog -> verifying-google-genai-refresh

Refresh completo del repo

Cuando hay que sincronizar cards, indices y mapa a la vez. Esta ruta siempre incluye el Topic Gap Discovery subagent.

using-google-genai-refresh -> Topic Gap Discovery subagent -> tracking-google-genai-lifecycle -> updating-google-genai-catalog -> verifying-google-genai-refresh

Solo comprobacion final

Cuando ya hubo cambios y solo queda garantizar coherencia interna.

using-google-genai-refresh -> verifying-google-genai-refresh
