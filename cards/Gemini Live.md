# Gemini Live

## TL;DR

The real-time spoken Gemini experience for live conversation, camera sharing, and screen sharing. It fits when the user interaction is voice-first and situational rather than standard text chat.

## What It Is

Gemini Live is the real-time interaction surface inside the consumer Gemini experience.

It is distinct from `Gemini Live API`, which is the developer-facing streaming layer. Here the value is end-user assistance through natural spoken interaction, with live context from the device camera or screen.

## When To Use It

- When the user wants spoken back-and-forth instead of typed chat.
- When camera or screen context improves the interaction.
- When the workflow is personal assistance, brainstorming, or situational help on a device.

## When It Is Not The Best Fit

- If the task is mainly typed chat, use `Gemini Web App` or `Gemini Mobile App`.
- If the need is programmable real-time interaction, use `Gemini Live API`.
- If the workflow centers on source-grounded notes and citations, use `NotebookLM`.

## Key Capabilities

| Capability | Why It Matters |
| --- | --- |
| Real-time spoken interaction | Makes Gemini feel conversational rather than prompt-by-prompt |
| Camera and screen context | Extends assistance beyond text-only inputs |
| Consumer-facing surface | Fits users who want immediate help without building anything |
