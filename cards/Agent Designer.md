# Agent Designer

## TL;DR

The low-code visual builder for designing and testing agents. It fits when you want to reason about agent structure visually before or alongside code.

## What It Is

Agent Designer acts as the low-code visual builder for designing and testing agents within `Agents Workflow Orchestration`. The suite is easiest to understand as a lifecycle: design, build, deploy, operate, and reuse agent components.

## When To Use It

- When you want to reason about agent structure visually before or alongside code.
- When the interaction model you need looks more like `Visual agent builder` than like a generic one-size-fits-all surface.
- When the main fit is in the `Agents Workflow Orchestration` block rather than in another part of the ecosystem.
- When you want to solve one concrete part of the workflow without adopting a broader platform up front.

## When It Is Not The Best Fit

- If your use case is closer to `Agent Development Kit`, start there instead.
- If your use case is closer to `Vertex AI Agent Engine`, start there instead.
- If your use case is closer to `Agent Garden`, start there instead.
